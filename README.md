# 7 Days of Code — Fullstack

Repositório com desafios práticos organizados por área de desenvolvimento.

## Estrutura

- frontend/: HTML, CSS, JavaScript, React
- backend/: APIs, Node.js
- dados/: bancos de dados, queries, modelagem
- mobile/: aplicações móveis
- outros/: experimentos diversos

## Objetivo

Construir domínio prático através de implementação direta de desafios,
com foco em código e evolução técnica.
